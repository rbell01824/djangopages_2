DjangoPages: Data Driven Pages

DjangoPages lets technical end user create and manage rich database driven pages with graphs, tables, forms, and other elements using Django's admin interface and a powerful set of easy to use widgets. It's a bit like Django's flatpages on steroids.

A DjangoPage can contain:

  • Bootstrap3 grid layouts.
  • Database queries and include the results on the page.
  • Arbitrary Python code and include the results on the page.
  • Easy to create charts and graphs built on top of Highcharts.
  • Chart and graph export to a variety of file formats.
  • Forms and filters.
  • Markdown content.
  • CSV export of table and chart data.

DjangoPages allows end users to create pages that used to require programmers. It is based on bootstrap3, font-awesmome & gliphicons, jquery, chartkick, highcharts, taggit, ace, and other leading technologies.
